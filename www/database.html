eZ 프레임워크는 HTML, CSS, Java Script 등을 템플릿으로 사용할 수 있으며, 복잡한 템플릿 문법이 없습니다. 단지 swap, append 등의 함수를 제공합니다. 정규표현식에 친숙하다면 이를 사용해서 문자열을 조작할 수도 있습니다. eZ 프레임워크는 자신의 웹서버로 돌릴 실행파일을 직접 만듭니다. 아파치나 톰캣 같은 덩치 큰 서버 프로그램을 돌릴 필요가 없습니다. eZ 프레임워크는 https 암호화와 가상 환경을 제공하는 미들 서버와 웹 서버로 나뉘어 있습니다. 미들서버는 프레임워크에서 곧장 사용할 수 있는 기성품의 형태로 제공합니다. 개발자는 웹 서버만을 개발하면 됩니다. 이 미들서버를 사용하면 복잡하게 설정을 할 필요가 없이 쉽게 암호화 된 웹사이트를 구현할 수 있습니다. 이 미들서버는 만약 아파치 서버와 외부 브라우저 사이에 위치시킨다면 아파치 서버의 설정에 아무런 변경 없이 암호화를 제공할 수 있을 정도로 유연합니다. 물론, eZ 프레임워크로 개발한 웹 서버와 브라우저 사이에 위치시킨다면, Stateful한 편리한 개발 환경까지 제공합니다.
