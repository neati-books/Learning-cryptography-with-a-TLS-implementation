Install

Web Server is mainly operated on *nix systems. 
> git clone https://github.com/zetapark/ez
> sudo apt install libmysqlcppconn-dev libjsoncpp-dev nettle-dev libgmp-dev libgmpxx4ldbl
> cd ez
> make
libzeta.so, middle.tls.x and site.src.x will be generated. middle.tls.x is a middle server and site.src.x is a web server. libzeta.so is a dynamic library for our programs. Let's run executables. 
> middle.tls.x &
> site.src.x &
Access https://localhost:4433 with any browser. You will see a very simple form. Let's click button. The value in the text field will change. It is from server side. 
Directory Structure
.
├── database : mysql database utility class
├── obj : object files are genearated here
├── util : log, command line parameter utility
├── tcpip : tcpip, website header
├── tls : middle server and tls
├── src : webserver source
└── www : webserver template
You just need to touch two directories that is 'src' and 'www'. 'src' directory is where server side programming sources should be. And web templates are in 'www' directory.
