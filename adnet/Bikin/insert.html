웹 페이지에 배너 삽입하는 법

        How to insert AdNET banner into a web page

다음의 코드를 웹페이지에 삽입하세요.

Please insert code as follows.

<head>
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
<script src=https://ad.zeta2374.com/ad.js></script>
</head>
<body>
<img src='default image' id=ad myid='your id' type=square category='computer programming' interval=20 width=100%>
</body>

AdNET은 JQuery를 사용합니다. AdNET의 자바스크립트를 인용하기 전에 JQuery를 참조해야 합니다. 그리고 원하는 위치에 img 태그를 id를 ad로 하여 삽입하세요. 이 이미지가 들어가는 곳에 광고가 표시될 것입니다. 어떤 페이지에 어떤 위치에 삽입할 것인지, 크기를 어떻게 할 것인지 스스로 정하면 됩니다. 이 이미지는 패널 속에 넣을수도 있고, 모바일 혹은 데스크탑에서 화면 크기에 따라 다르게 표현되도록 할 수도 있습니다.
img 태그의 id, myid, type은 항상 지정해야 하고, src, category, interval은 생략할 수 있습니다. src는 웹 사이트의 문제로 광고가 받아지지 않거나, 광고 차단 소프트웨어를 사용할 경우 대체로 보여질 이미지입니다. 그러므로, 이 소스는 자신의 웹사이트에서 항상 액세스할 수 있는 것으로 하는 것이 좋고, type과 크기, 비율이 일치해야 합니다. interval은 광고가 갱신될 주기입니다. 초단위로 셋팅할 수 있으며, 최소는 10초입니다. width 혹은 height 요소는 마음대로 정할 수 있습니다. 단, AdNET의 배너는 같은 type이라 해서 크기가 똑같은 것이 아니라, 가로, 세로 비율만이 똑같은 것이므로, 이를 고려하여야 합니다. 부트스트랩 등을 사용하여 responsive한 웹페이지를 디자인하거나, 심지어는 모바일에서는 광고가 사라지게 할 수도 있습니다. 웹페이지에 적절한 범위를 광고가 점유하게 잘 조절하십시오. 그러나, 광고를 보여주는 것이 아니라, 광고를 클릭한 횟수가 중요하다는 것을 잊지 마십시오.

AdNET uses JQuery. So you should reference JQuery before including ad.js. And finally include img tag with id=ad. Banner will be displayed where this img is placed. When and where your image will be displayed is all upon your command. This image can be displayed inside a panel or can be displayed differently according to screen size with responsive libraries like bootstrap.
You should specify id, myid, type of img tag. category, interval can be omitted. src is a image that will be displayed when there is some problem with showing the ad images. This is for the cases like using ad-block software or web malfunction. This image will be displayed as default for those cases. So, you had better let a image that is always accessible from site. And this image should be compliant with the type in size and width x height ratio. Interval is the time period to refresh the banner. You can control it by seconds. Minimum is 10 seconds. You can arbitrarily decide width or height just as you did with your images. But, the size of AdNET banners are different even if the type is the same. Only the width x height ratio is always the same. Please take this into account. You can even use Bootstrap to make the banner responsive or disappear on mobile devices. Control these to make your banner occupy proper portion. But we remind you that click induce matters not the view induce.

AD type

AD type은 다음의 네 가지가 있습니다. 4종의 배너 중 지정된 배너가 보여질 것입니다.

There are 4 types. Designated banner among 4 types will be displayed.

  • leader : 728 x 90
  • square : 300 x 250
  • mobile : 320 x 50
  • sky : 160 x 600

category

카테고리를 지정하면, 지정한 category에 해당되는 배너가 더 자주 표시될 것입니다. category를 지정하지 않으면, 모든 배너가 차별없이 표시됩니다. category는 다음의 타입이 있으며, 여러 개를 동시에 지정할 수 있습니다. 이 때 각 카테고리의 사이는 스페이스로 띄어야 합니다.(i.e. category='education travel food')

Those banners that match designated category will be displayed more. If you do not designate the category, then banner category will be ignored. There are category types as follows. You can specify multiple categories together. In that case, you should add a space between each categories.(i.e. category='education travel food')

  • computer
  • programming
  • game
  • gadgets
  • education
  • shopping
  • travel
  • food
  • entertainment
  • society
  • fashion
  • health
  • finance
  • others
