C++ side Programming

#include"tcpip/server.h"
#include"tcpip/website.h"

int main(int ac, char** av) {
	WebSite my_site;
	my_site.init("site_html");//html template directory
	Server sv{2000};//port number
	sv.start(my_site);//go infinite loop
}

compile using make.
'site_html' is relative directory name to our executable.
Go to the directory where our executable is.
Make directory named site_html.
Locate index.html file there.
run the executable.
Now, if we access localhost:2000 with any webbrowser,

make을 사용해 컴파일 하세요.
'site_html'은 현재 실행 파일에 대한 상대 경로명입니다.
실행 파일이 있는 디렉토리로 가세요.

실행 파일을 실행시킵니다.
